# Libasm
The aim of this project is to become familiar with assembly language by recreating some basic functions

This project helped me better understand registers and the ABI calling convention.
It also helped me understand how a stack frame works and how it should be managed.
I already had some background knowledge of what the project required because I had previous experience through GameHacking projects.
